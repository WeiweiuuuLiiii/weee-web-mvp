# WEEE Web MVP

Live Demo: https://weiweiuuuliiii.github.io/weee-web-mvp/

Product-style web MVP for a grocery ordering funnel:
Search → Out-of-stock recovery → Cart → Checkout review.

## Repo structure
- `index.html` — entry page
- `src/` — app logic (state, events, rendering)
- `styles/` — styling
- `docs/` — spec, event schema, edge cases

## Run locally
Open `index.html` in any modern browser.

## Engineering
- Deterministic UI states
- Lightweight event logging (see `docs/events.md`)
- Edge-case checklist (see `docs/edge-cases.md`)

## Notes
This is a prototype MVP intended for rapid iteration and usability validation, with a clean path to expand into a full-stack implementation.
Deployed via GitHub Pages.
